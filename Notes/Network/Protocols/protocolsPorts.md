# Protocols and Port Numbers

## Default Port Numbers

| Protocol | Default Port Number |
|----------|---------------------|
| HTTP     | 80                  |
| SMTP     | 25                  |
| POP3     | 110                 |
| IMAP     | 143                 |

## Secure Versions (over TLS)

| Protocol | Default Port Number |
|----------|---------------------|
| HTTPS    | 443                 |
| SMTPS    | 465 and 587         |
| POP3S    | 995                 |
| IMAPS    | 993                 |
